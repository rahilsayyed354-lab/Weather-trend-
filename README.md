Weather Pattern Analysis Dashboard

Project Overview

Weather Pattern Analysis Dashboard is an interactive data analytics web application developed to analyze and visualize weather data through modern charts, climate metrics, and comparative insights. The dashboard helps users understand temperature variations, rainfall distribution, humidity patterns, and seasonal climate trends using visually appealing and responsive data visualizations.

The project is designed using HTML, CSS, JavaScript, and Chart.js, providing a professional analytical interface that enables users to monitor weather conditions, compare climate data across multiple cities, and identify seasonal patterns through interactive visualizations.

How to Run

1. Download or clone this repository from GitHub.
2. Open the project folder on your computer.
3. Locate the HTML file.
4. Double-click the HTML file or open it using any modern web browser such as Google Chrome, Microsoft Edge, Firefox, or Brave.
5. Ensure an internet connection is available for loading Chart.js and Google Fonts from CDN.
6. The dashboard will launch automatically and display temperature trends, rainfall analysis, humidity distribution, and city-wise climate insights.
